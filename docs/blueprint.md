# **App Name**: CampusConnect

## Core Features:

- Landing Page Animation: An animated college logo intro and glassmorphism/gradient background to engage users.
- Lost Item Posting: Allow students to post lost items with image upload, descriptions, and category selection.
- Item Search and Filter: Enable students to search and filter items by type, date, and location on the dashboard.
- Smart Matching: Show possible matches for lost items using mock data and UI elements.
- Gamified Helper Rewards: Implement a helper rewards system with progress bars and level badges (Bronze, Silver, Gold).
- Campus Map Integration: Display a visual-only campus map with markers for lost and found item locations.
- AI Item Suggestion: Suggest possible lost item names or descriptions using a generative AI tool.

## Style Guidelines:

- Primary color: #577399 (a muted, sophisticated blue) to give a calm, reliable impression appropriate for a college setting.
- Background color: #F2F4F8 (a very light, desaturated tint of the primary blue).
- Accent color: #83BA74 (a soft, springlike green) for interactive elements and AI suggestions, to set it apart without disrupting the primary impression.
- Font pairing: 'Space Grotesk' (sans-serif) for headlines and 'Inter' (sans-serif) for body text.
- Use 'Lucide-react' icons for a consistent, modern look.
- Card-based layout for displaying items in a responsive manner.
- Smooth UI transitions and hover glow effects using Framer Motion.
